## 5) Storyboard (7 Shots für ~30 s — je einzeln generieren & schneiden)

1. **Hook (3 s):** Pure black. A single orange ember ignites; the KatoOS shield flares into existence
   with an orange light-burst and ember particles, rim-lit, slow rotate. *Overlay:* „KatoSync".
2. **Problem (4 s):** A cold blue-grey cloud of scattered documents, sticky notes, PDF icons and code
   files swirling in chaotic disarray in deep charcoal space. Cluttered, overwhelming.
3. **Sog (5 s):** An orange tractor-beam pulls the scattered files toward the glowing shield; as they
   pass through, the cold blue tint warms to orange, chaos becomes ordered streams. Slow push-in.
4. **Produkt-UI (5 s):** Smooth morph into a hyper-clean dark macOS interface — a dashboard "cockpit"
   with animated orange donut + bar charts, a live activity feed, an "Action Queue"; a cursor approves
   a task with a satisfying orange glow-pulse. Cinematic screen aesthetic floating in 3D, slight parallax.
5. **Ergebnis (5 s):** The approved task spawns finished artifacts that elegantly assemble: a tailored
   cover letter and CV slide into place, a briefing card with charts, a code diff — each tagged with a
   small orange checkmark. Premium, satisfying.
6. **Vertrauen (3 s):** A frosted shield-with-lock icon, orange glow, encrypted data particles orbiting.
   *Overlay:* „Zero-Knowledge · Du entscheidest".
7. **Logo-Payoff (4 s):** Camera pulls back to the KatoOS shield glowing on black, ember sparks settling.
   *Overlay:* Logo + Tagline.

## 6) Style-Bibel

- **Palette:** Schwarz/Anthrazit `#0D0F13` + Orange `#FF6B35` / `#FF8C3A`; sparsam Grün `#34D399` für
  „erledigt"-Häkchen. Kalt-Blau nur für „Chaos/Problem".
- **Logo:** das faceted Orange-Schild-„K" mit Glut/Glow — immer der visuelle Anker.
- **Typo-Overlays:** clean Sans (SF Pro / Inter), weiß, viel Negativraum, dezent.
- **Motion:** langsame, selbstbewusste Kamera (push-in, pull-back), weiche Übergänge, ein „Snap" pro
  Approve-Moment. Kein hektischer Schnitt.
- **Audio:** dunkles cinematic Electronic, tiefer Bass-Puls baut auf → aufsteigender Synth; „Whoosh"
  beim Sog, „Click/Chime" beim Approve, warmer Swell beim Logo. (Vercel/Linear-Promo-Vibe.)
